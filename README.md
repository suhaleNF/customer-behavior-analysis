# Customer Behavior & Revenue Analysis

## 📊 Overview
This project analyzes customer behavior and revenue patterns using SQL, Excel, and Tableau.

## 🛠 Tools Used
- SQL
- Excel (Data Cleaning)
- Tableau (Visualization)

## 🧹 Data Cleaning
- Removed duplicates
- Handled missing values
- Standardized city and category fields
- Created revenue column

## 📈 Analysis Performed
- Monthly revenue trend
- Revenue by city
- Top customers by revenue

## 💡 Key Insights
- Revenue fluctuates across months
- Revenue is concentrated in key cities
- Top customers contribute a large share of revenue
